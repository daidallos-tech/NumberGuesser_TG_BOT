# NumberGuesser_TG_BOT

# The Rules📋

The bot guesses the number from 1 to 100!
You have 7 attempts to guess secret number!
If you do it - you'll win.

# The commands ❗️

/start - restart bot
/help - about game and commands
/stat - your statistic
/cancel - to stop game in game

If you want to play just type 'Start', 'Go', 'Play'
To exit just type 'Exit', 'No', 'Stop' - in game type - /cancel

It works pretty simple now!
